# 2024SQTP
本项目旨在通过优化浙江大学校园内电瓶车充电桩的数量与布局，改善充电流程，解决学生在日常出行中等待时间长、充电充不满等充电难题，提升校园基础设施的服务水平，缓解同学间的矛盾，提升出行便利性，从而增强学生的归属感和满意度。 为解决这一问题，我们小组将进行实地考察和调研，收集充电需求数据，根据调研结果，考虑人流量、停车区等因素，制定充电桩增设与改建方案。同时我们将与校方和相关部门积极沟通交流，协调校园内各方资源，在同学间通过各种方式大力宣传，项目实施后实时跟踪监测进度、定期调整方案，力求找到最好的方式来优化充电桩的布局分布和改善充电流程，解决同学们的充电难题，提升学生的满意度和校园体验，同时使学校内的秩序更加规范，营造便利和融洽的校园环境。
